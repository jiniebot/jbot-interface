## Key Visual Differences

### Typography Scaling

| Element | Desktop | Tablet | Mobile | Small Mobile |
|---------|---------|--------|--------|--------------|
| Hero Title | 3rem | 2.5rem | 2rem | 1.75rem |
| Section Title | 2.5rem | 2rem | 1.75rem | 1.5rem |
| Body Text | 1rem | 0.9375rem | 0.9375rem | 0.875rem |
| Button Text | 1rem | 0.9375rem | 0.9375rem | 0.875rem |

### Spacing

| Element | Desktop | Mobile |
|---------|---------|--------|
| Panel Padding | 3-4rem | 1.5-2rem |
| Button Padding | 1rem 2.5rem | 0.875rem 2rem |
| Section Gap | 2rem | 1rem |
| Content Margin | 2rem | 1rem |

### Touch Targets

| Element | Desktop | Mobile Minimum |
|---------|---------|----------------|
| Button | Any size | 44x44px |
| Icon Button | 48x48px | 44x44px |
| Toolbar Item | 80x80px | 60x80px |
| Nav Link | Any | 44px height |

### Layout Changes

**Desktop:**
- Side-by-side panels
- Horizontal navigation
- Floating overlays
- Multi-column grids

**Mobile:**
- Stacked layout
- Vertical navigation
- Full-screen overlays
- Single column

## Interaction Differences

### Hover vs Touch

**Desktop (Hover):**
```css
button:hover {
  transform: translateY(-2px);
  box-shadow: 0 8px 20px rgba(0,0,0,0.3);
}
```

**Mobile (Touch):**
```css
button:active {
  transform: scale(0.97);
  background: rgba(255,255,255,0.25);
}
```

### Navigation

**Desktop:**
- Click anywhere
- Hover for tooltips
- Right-click menus
- Keyboard shortcuts

**Mobile:**
- Touch with minimum 44px target
- Long-press for context
- Swipe gestures (native)
- Virtual keyboard aware

## Landscape Mode Changes

**Portrait Mobile (390x844)**
```
Full height sections
Vertical scrolling
```
